# Sistemi Operativi

Questa repository contiene i miei appunti personali per il corso di Sistemi Operativi (Anno Accademico 2024/2025) presso l'Università di Catania.

Il materiale include:
- Appunti presi durante le lezioni
- Integrazioni e chiarimenti tratti dal libro di testo
- Esercizi di pratica sui principali argomenti del corso
- Tentativi di svolgimento di prove d’esame passate

## Argomenti principali

Gli esercizi e gli esempi riguardano principalmente:

- Gestione dei file
- Gestione delle directory
- Memory mapping (`mmap`)
- Thread
- Processi

## Disclaimer

Questi appunti sono materiale di studio personale e potrebbero contenere imprecisioni o errori.

Le soluzioni delle prove d’esame presenti nella repository rappresentano **miei tentativi di risoluzione** e **non devono essere considerate soluzioni ufficiali**.
